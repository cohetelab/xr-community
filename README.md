# XR 커뮤니티 (Demo)

XR 사용자 커뮤니티 사이트의 **메인 페이지 클론 프로토타입**입니다.
PICO 커뮤니티 같은 일반적인 XR 포럼의 레이아웃/기능 구조를 참고해 직접 제작했으며,
브랜드명·로고·이미지·텍스트는 모두 자체 플레이스홀더입니다.

## 구성

```
xr-community/
├─ index.html          # 메인 페이지 마크업
├─ assets/
│  ├─ styles.css       # 디자인 토큰 + 전체 스타일 (라이트 테마, 블루 액센트)
│  └─ app.js           # 히어로 캐러셀, 탭 전환, 피드/랭킹 렌더링
└─ README.md
```

## 미리보기

빌드 도구 없이 바로 열립니다.

- 방법 1: `index.html` 더블클릭
- 방법 2: 로컬 서버 (권장)
  ```bash
  npx serve .        # 또는
  python -m http.server 8000
  ```
  → http://localhost:8000

## 구현된 섹션 (메인)

- 상단 고정 헤더 (로고 · 메뉴 · 검색 · 로그인/회원가입 · 모바일 햄버거)
- 히어로 배너 캐러셀 (자동 슬라이드 + 화살표 + 도트)
- 카테고리 타일 6종
- 커뮤니티 피드 (최신/인기/주목 탭, 게시글 카드)
- 우측 사이드바 (환영 카드 · 인기글 랭킹 · 인기 태그 · 현황 통계)
- 푸터
